Covid 19 Status

CovidAPI provides a simple API for COVID-19 pandemic data. Data sources for COVID-19 are disparate and difficult to consume, consisting of PDF files, CSV files and unexportable maps. This API seeks to provide users with the ability to easily consume COVID-19 pandemic data as JSON and use the data to build meaningful applications in support of COVID-19 relief efforts. Usage of this API is free. For any questions, comments or concerns, please email me at me@blackiq.ir.

The primary data source for the CovidAPI is the 2019 COVID-19 Data Repository by Johns Hopkins CSSE. This data repository leverages several different data sources. The WHO, DXY, NHC, CCDC, USCDC, ECDC and more. Data is refreshed once a day around 23:59 UTC and as soon as data is published to Johns Hopkins CSSE sources it is ingested by CovidAPI. The World Health Organization releases a daily situation report with information on the COVID-19 pandemic.
